## :gear: Technologies

This project was developed during Ignite, course of the React Rocketseat, with the following technologies:

  - [ReactJS](https://reactjs.org/)
  - Next JS
  - [Typescript][ts]
  - Styled Components
  - Jest

## :computer: How to run

  ```bash
    # clone repository
    git clone

    # enter web folder
    cd project-name

    # download dependencies
    yarn

    # running web project
    yarn run dev
  ```
